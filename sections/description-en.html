EVI: Description back to ToC

Figure 1. shows how the EVI ontology is used to represent the provenance of a scientific experiment. A scientific experiment is represented as a p-plan:Plan which consists of several steps. Each step of a scientific experiment has input and output variables. The ontology also describes how the instruments, materials and settings are linked to the experiment.
