EVI: Description back to ToC

Figure 1. shows the EVI semantic model built using the classes and object properties. The core PROV ontology concepts used in EVI are Entity, Activity, and Agent, with two suc-classes Person and Organization. The object properties in EVI are used to establish relations among instances, most of which are of type Representation. Computation and DataAcquisition are Activities performed on instances of other Representations, Software or Services.
